## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

### npm install
```bash
npm install
```

## Running the app

### Run

```bash
# development
docker compose up api
```

see start node scripts
```bash
# for linux
source env.sh
# for other us `[local.env](local.env)`
npm run start:debug

# or
npm run
# watch mode
# npm run start:dev
# ...
```
